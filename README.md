# Currency Converter

A simple web-based currency converter that fetches real-time exchange rates.

## Usage

1. Open `index.html` in your web browser.
2. Enter the amount you want to convert.
3. Select the source currency from the first dropdown.
4. Select the target currency from the second dropdown.
5. Click the "Convert" button.
6. The converted amount will be displayed below.

## Features

- Supports USD, EUR, and UAH currencies.
- Fetches live exchange rates from exchangerate-api.com.
- Simple and responsive design.

## Troubleshooting

- **No result after clicking Convert**: Check your internet connection, as the app needs to fetch exchange rates from an external API.
- **Invalid amount**: Ensure you enter a positive number.
- **Error message**: If there's an issue with the API, try again later or check the console for details.

## Files

- `index.html`: The main HTML structure.
- `style.css`: CSS for styling the page.
- `script.js`: JavaScript for the conversion logic.
