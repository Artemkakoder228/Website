Як встановити Програму

Дотримуйтесь цих простих кроків для швидкого старту

01

Завантаження інсталятора

Натисніть кнопку "Завантажити файли програми" під цією інструкцією. Переконайтеся, що вам завантажився zip файл.

Currency Converter
02

Встановіть відповідне програмне забезпечення

Щоб програма працювала коректно також потрібно завантажити додаткове ПЗ, натиснувши на кнопку Завантажити ПЗ. "Встановлення" -> "Провести процедуру встановлення .Net".

03

Перевірка роботи програми

Після встановлення програми запустіть її і переконайтесь, що вона працює коректно. Перших 3 банки та біржі які встановлені мають працювати з коробки

04

Обов'язкова умова роботи

Файл який ви завантажили потрібно лише розархівувати і запустити програму з всіма допоміжними файлами які є в ній.

05

Вибір та налаштування API

Для отримання актуальних курсів ви можете використовувати вбудовані API або додати нові. Повний список підтримуваних сервісів (NBU, Monobank, Binance тощо) знаходиться на спеціальній сторінці.
